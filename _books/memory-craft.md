title: Memory Craft
subtitle: Improve your memory using the most powerful methods from around the world
author: Lynne Kelly
---

Reading Memory Craft is like wondering down a street you've not seen in a while, everything seems so familiar... and a little alien. You'll find things you've done for years, things you've never heard of... and things that bring a bit of fun to life. All while learning how to use your memory.

There are many how-to books that teach a curriculum. This book teaches a way of life.
